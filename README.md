# 🌦 Weather Dashboard

A responsive Weather Dashboard Web App that displays real-time weather information.

## 🚀 Features

- Search weather by city
- Get weather using current location
- 5-day forecast
- Hourly temperature chart
- Weather animations
- Progressive Web App support

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- Chart.js

## 🌐 API Used

Free weather API from Open-Meteo  
https://open-meteo.com
